Projects

Through my undergraduate and graduate studies, I have done several projects including course projects, BSc. project and projects done for competitions. Below, I provided a summary of each project accompanied by its link on GitHub.

  • Simulation of a non-cooperative broadcast algorithm (Current)

    In this project, I am simulating a non-cooperative broadcast algorithm for Wireless and Ad-Hoc Sensor Networks presented in the paper "An Exponential Improvement on the MST Heuristic for Minimum Energy Broadcasting in Ad Hoc Wireless Networks". To do so, I needed to use a programming language that was fast and efficient. Therefore, I chose to code in C++ language.

  • Phone Book

    This is a classic phone book implementation having most of the capabilities of a typical phone book program (application) such as adding, deletting, updating, or searching for contact entries. Through the implementation of this project, I learned

    • Object Oriented Programming (OOP)
    • Ruby programming language
    • Regular Expressions
    • Model-View-Controller (MVC) paradigm

  • Advancement of a Web Server on an ARM-based Embedded System (BSc. project)

    This project was done in cooperation with my friend, Seyyed Farshad Kazemi at Ferdowsi University of Mashhad. We started with simple web server code, and developed it on an embedded system which used an ARM processor (AT91SAM7x256). the code of the simple web server was provided by the system (board) manufacturer. The project involved two main parts of C++ coding and web design. For the C++ coding, it required manipulation of processor's registers, which required familiarity with the board's schematic and datasheets of processor and other ICs that were used on the board. In addition, we developed the server-side of web server to respond to the requests sent from users. For the front-end, we developed web designs for the users and added extra pages to represent the real-time data that was generated by the sensors that were connected to the board.

  • FPGA Design Contest (Blockus Contest)

    This is project done through the 1st National Digital System Design Contest of Iran. This Contest was consisted of two categories: 1) FPGA Design Contest and 2) Hardware-Software Co-design Competition. We participated in the fisrt category as a 3-member team including to Electronics engineer and one Software engineer. The contest was a live event of a two-player blokus-duo game where the contestants designed hardware modules on FPGA to perform as one player of the game. The contestant designs competed against one another in a live league on the final day of the contest. The rules applied to the competition where similiar to the rules in ICFPT 2013 Design Competition: Blokus Duo. We had our design written in VHDL language and the FPGA board that was used in the competition was Altera DE2.
